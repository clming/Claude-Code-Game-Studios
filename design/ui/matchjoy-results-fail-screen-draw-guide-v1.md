# MatchJoy Results Fail Screen Draw Guide v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the layer-by-layer drawing order for the fail results screen in Figma.

## Target Frame

- frame: `MatchJoy / iPhone17 / ResultsFail / v1`
- size: `393 x 852`

## Drawing Order

### Layer 1. Base InLevel Screen

Start from the approved in-level screen composition.

Rule:

- keep the board visible enough to preserve context
- do not let the background become muddy

### Layer 2. Overlay Atmosphere

Add:

- warm muted veil
- restrained coral pressure tint

Rule:

- failure should feel branded and gentle, not punitive or system-like

### Layer 3. Results Card Base

Add:

- same shared silhouette as the win card
- adjusted fail-state accent strip
- warm cream card body retained

Rule:

- loss should reuse the same premium structure as win

### Layer 4. Results Card Content

Add in this order:

1. overline
2. chapter / level badge
3. fail headline
4. performance summary
5. support detail
6. footer

Rule:

- the card should communicate disappointment without collapsing into alarm UI

### Layer 5. Fail-State Accent

Add:

- minimal badge or star treatment if needed
- coral accent glow or edge treatment only

Rule:

- avoid trophy-like reward treatment
- avoid hard error-icon language

### Layer 6. Final Polish

Add:

- subtle tonal balancing
- soft depth cleanup
- final readability check for all copy

## Review Questions

1. Does the fail state still feel premium?
2. Does it avoid generic red-alert UI language?
3. Is the structure clearly related to the win card?
4. Is the emotional tone clear without becoming harsh?
