# MatchJoy UI Asset Start Order v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the exact first assets that should be created when UI art production begins.

## Start Order

### Step 1

Create:

- `hud-card-bg-v1`

Reason:

- the HUD card sets the language for the top information module

### Step 2

Create:

- `board-shell-outer-v1`

Reason:

- the board shell defines the hero frame of the whole screen

### Step 3

Create:

- `board-tray-inner-v1`

Reason:

- the playable area needs a stable inner surface before later polish

### Step 4

Create:

- `results-card-bg-v1`

Reason:

- win / fail should share one premium base before diverging

### Step 5

Create:

- `board-stage-pill-base-v1`
- `board-status-badge-base-v1`

Reason:

- these two define how board state reads in the shell

### Step 6

Create:

- `results-accent-strip-v1`
- `hud-accent-strip-v1`

Reason:

- these push the screens from structural to branded

## Start Order Rule

If production time is limited, the first four assets are the minimum viable visual batch.

Those four assets should be reviewed together before symbol sets and polish assets begin.
