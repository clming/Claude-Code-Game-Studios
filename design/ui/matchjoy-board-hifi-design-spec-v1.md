# MatchJoy Board High-Fidelity Design Spec v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the first-pass high-fidelity art direction for the board hero module.

## Design Intent

The board shell should feel like the hero work surface of a premium candy workshop.

It should read as:

- the main tray,
- the center of player attention,
- stable and trustworthy,
- slightly luxurious but still playful.

## Material Direction

The board frame should feel like:

- lacquered confection tray,
- polished candy counter trim,
- soft glazed rim with restrained shine.

Preferred treatment:

- warm cream or beige body
- deeper caramel outer rim
- soft syrup highlight on top edges
- subtle inner shadow for grid separation

Avoid:

- heavy metallic game-board frames
- dark fantasy ornamentation
- sterile flat rectangles

## Shape Language

- thick outer frame
- rounded corners
- soft capsule stage pill
- compact right-side badge card
- near-square inner play area

## Typography Hierarchy

### Board Title

Role:

- identify the current work surface

Style target:

- strong but secondary to the grid itself
- warm, bold, rounded display style

### Stage Pill

Role:

- communicate board state at a glance

Style target:

- compact capsule
- bolder contrast than subtitle
- short all-caps or tight title case

### Subtitle

Role:

- support line only

Style target:

- small and quiet
- readable but low emphasis

### Status Badge

Role:

- summarize pace, pressure, goal progress

Style target:

- compact, tidy, visibly separate from title row

## Color Direction

Base:

- cream tray body
- caramel frame edging
- brown text support

Accents:

- syrup orange for stable-active state
- deeper coral-orange for pressure state
- warm gold highlights for polished clarity

State rule:

- stage color and pressure color can both influence the shell
- pressure should increase warmth and urgency
- but the board should still remain readable and premium

## Decorative Elements

Allowed:

- subtle top-edge gloss
- tiny candy-cut corner highlights
- low-density sprinkle or sugar dust accents outside the play area
- one compact badge treatment

Avoid:

- dense ornaments intruding into the grid
- multiple competing badges
- oversized chrome that shrinks play readability

## Motion Direction

The board shell should support:

- stage pill snap or glow on state change
- tiny accent shimmer when entering resolve state
- subtle pressure-color shift when the run gets tight

Avoid:

- large frame pulses
- moving decorative clutter around the grid

## Asset Notes

Likely assets needed:

- board outer frame
- board inner frame or tray base
- stage pill backgrounds
- status badge background
- minor gloss / edge overlays

## Figma Build Notes

For the first high-fidelity frame:

- keep the inner playable area generous
- establish the board shell first before decorating it
- evaluate the shell at grayscale before committing full color
- confirm that the grid still dominates visually after polish is added
