# MatchJoy Figma Initial Frame Checklist v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the first set of frames that should be created in the MatchJoy Figma file.

## Goal

This checklist gives the exact first-pass frame list for the Figma file.

1. `MatchJoy / References / HUD`
2. `MatchJoy / References / Board`

4. `MatchJoy / References / RouteSummary`

### Page 04_ExportPrep

2. `MatchJoy / Export / ResultsWin / v1`
3. `MatchJoy / Export / ResultsFail / v1`
4. `MatchJoy / Export / ComponentsOverview / v1`

## First Creation Order

Build frames in this order:

1. low-fidelity full screen
2. hi-fi in-level
3. hi-fi results win
4. hi-fi results fail
5. component frames
6. export prep frames

## Minimum Completion Rule

The Figma file is not ready for review until these three frames exist:

- `MatchJoy / iPhone17 / InLevel / v1`
- `MatchJoy / iPhone17 / ResultsWin / v1`
- `MatchJoy / iPhone17 / ResultsFail / v1`
