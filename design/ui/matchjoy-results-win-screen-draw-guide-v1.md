# MatchJoy Results Win Screen Draw Guide v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the layer-by-layer drawing order for the win results screen in Figma.

## Target Frame

- frame: `MatchJoy / iPhone17 / ResultsWin / v1`
- size: `393 x 852`

## Drawing Order

### Layer 1. Base InLevel Screen

Start from the approved in-level screen composition.

Rule:

- dim it only lightly
- keep the board readable behind the overlay

### Layer 2. Overlay Atmosphere

Add:

- soft warm veil over the play screen
- optional restrained sparkle haze

Rule:

- do not darken the screen into a modal blackout

### Layer 3. Results Card Base

Add:

- centered premium card silhouette
- top accent strip or crest zone
- warm honey-cream card body

### Layer 4. Results Card Content

Add in this order:

1. overline
2. chapter / level badge
3. headline
4. performance summary
5. support detail
6. footer

Rule:

- headline and summary should lock first before stars or ornaments

### Layer 5. Reward Accent

Add:

- star row or star glyph treatment
- small sparkle accents
- optional soft reward burst behind headline or summary

Rule:

- keep celebration premium and controlled

### Layer 6. Final Polish

Add:

- highlight accents
- subtle shadow balancing
- glow cleanup

## Review Questions

1. Does the card feel branded rather than generic?
2. Does the win state feel warm and satisfying?
3. Is the board still visible enough behind the card?
4. Are stars and reward accents supporting hierarchy rather than replacing it?
