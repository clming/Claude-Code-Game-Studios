# MatchJoy InLevel Screen Draw Guide v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the layer-by-layer drawing order for the in-level hero screen in Figma.

## Target Frame

- frame: `MatchJoy / iPhone17 / InLevel / v1`
- size: `393 x 852`

## Drawing Order

### Layer 1. Background Atmosphere

Add:

- warm cream vertical background base
- very subtle candy-shop ambient gradient
- optional low-contrast sugar dust or confection pattern in corners only

Rule:

- background must support the board, not compete with it

### Layer 2. Top Metadata Support

Add:

- safe-area guide
- tiny route metadata support if needed

Rule:

- keep this zone visually light

### Layer 3. HUD Card Base

Add:

- one premium order-card shape
- accent strip
- soft highlight and depth treatment

### Layer 4. HUD Content

Add in this order:

1. overline
2. level title
3. goals block
4. move count block
5. pace block
6. footer ledger

Rule:

- confirm readability in grayscale before color polish

### Layer 5. Board Shell Base

Add:

- main board tray silhouette
- outer rim
- inner tray region

Rule:

- this layer must establish the board as the visual hero before any text is added

### Layer 6. Board Header Content

Add:

1. board overline
2. board title
3. stage pill
4. subtitle
5. status badge
6. footnote

Rule:

- the board frame must still dominate after header elements are added

### Layer 7. Placeholder Play Grid

Add:

- grayscale or early-color placeholder grid
- representative candy tiles
- enough to show density and readability

Rule:

- the grid should feel generous and central
- do not over-detail tile art in this first pass

### Layer 8. Micro Polish

Add only after hierarchy is approved:

- soft gloss hits
- tiny highlight accents
- subtle candy sparkle hints

Rule:

- polish should increase appetite, not add noise

## Review Questions

1. Does the board remain the hero?
2. Is the move count instantly visible?
3. Is the HUD card subordinate but still clear?
4. Does the screen feel premium without feeling crowded?
