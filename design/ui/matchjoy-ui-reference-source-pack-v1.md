# MatchJoy UI Reference Source Pack v1

> Status: Draft
> Last Updated: 2026-03-30
> Purpose: Define the first official reference-source pack for MatchJoy UI art design.

## Source Pack Goal

This source pack exists to make benchmark gathering faster and more consistent.

It is the approved first-pass list of places to pull reference from for:

- board framing
- HUD hierarchy
- results presentation
- icon / badge treatment
- motion tone

## Collection Rules

Only collect references that help answer a concrete design question.

Allowed categories:

- board framing
- HUD layout
- result / reward panel
- panel materials
- status chips / badges
- icon style
- motion / FX tone

Avoid collecting:

- random unrelated illustration inspiration
- off-theme fantasy UI
- overly generic mobile UI with no useful structure lessons

## Required Output From This Pack

The first pass should produce:

1. benchmark screenshots grouped by category
2. notes on what to keep / adapt / avoid
3. one shortlist of route-driving influences
4. one shortlist of patterns to reject
